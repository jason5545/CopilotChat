Privacy Policy for Copilot Chat

Effective date: April 11, 2026

Overview

Copilot Chat ("the App") is a native iOS multi-provider LLM chat client developed by Jui Chen Chien ("Developer"). The App connects to third-party AI providers (such as GitHub Copilot, OpenAI, Anthropic, Google, and others) selected by the user. The Developer does not operate any servers or intermediary services — all communications occur directly between the user's device and the chosen provider.

Data We Collect

The App does not collect, transmit, or store any personal data on Developer-controlled servers. Specifically:

  • No analytics or tracking: The App contains no analytics SDKs, advertising identifiers, or tracking mechanisms.
  • No server-side storage: Conversation history, settings, and credentials are stored exclusively on the user's device.
  • No user accounts: The App does not require creating an account with the Developer.

Data Stored On Your Device

  • API keys and tokens: Stored securely in the iOS Keychain (kSecAttrAccessibleAfterFirstUnlock).
  • Conversation history: Stored as JSON files in the App's sandboxed Documents directory.
  • Settings and preferences: Stored locally via UserDefaults.

Data Shared with Third Parties

When you use the App, your messages and any attached content (text, images) are sent directly to the AI provider you have selected. The Developer has no access to this data. Each provider has its own privacy policy governing how they handle your data:

  • GitHub Privacy Statement
  • OpenAI Privacy Policy
  • Anthropic Privacy Policy
  • Google Privacy Policy

Users are responsible for reviewing the privacy policies of their chosen providers.

MCP (Model Context Protocol) Servers

The App supports connecting to user-configured MCP servers. Authentication headers for these servers are stored in the iOS Keychain. Communication with MCP servers occurs directly from the user's device. The Developer has no access to MCP server connections or data.

Community Plugins (.cex)

The App supports user-imported community plugins that run in a sandboxed JavaScriptCore environment. Plugins may request permissions for network access, clipboard, and notifications. Plugin network requests are restricted to user-configured domain whitelists. The Developer does not distribute or control third-party plugins.

Photos and Media

The App allows users to attach photos to chat messages via the iOS photo picker. Photos are processed in-memory and sent to the selected AI provider as part of the conversation. Photos are not stored by the App beyond the active conversation.

Coding Mode and File Access

Coding Mode allows the App to access user-selected project folders via the iOS document picker. File access is granted through security-scoped bookmarks and persists only with explicit user permission. The App reads and writes files solely within the user-selected workspace directory.

Children's Privacy

The App is not directed at children under 13. The Developer does not knowingly collect personal information from children.

Data Retention

Since the Developer does not collect or store any user data, there is no data retention policy applicable to the Developer. Data stored on the user's device remains until the user deletes it or uninstalls the App.

Changes to This Policy

The Developer may update this privacy policy from time to time. Changes will be reflected by updating the "Effective date" above.
